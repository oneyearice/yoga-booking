## 测试背景

@coder 修复了首页按钮文字垂直居中问题，使用 flex 布局替代 line-height。

**Commit**: `39015bb` - fix: use flex for button text vertical center

---

## 代码审查

### 修改前
```css
.action-btn {
  min-width: 200rpx;
  height: 80rpx;
  line-height: 80rpx;      /* ❌ line-height 方式 */
  text-align: center;
  border-radius: 40rpx;
  font-size: 32rpx;
  font-weight: bold;
}
```

### 修改后
```css
.action-btn {
  display: flex;            /* ✅ flex 布局 */
  justify-content: center;  /* ✅ 水平居中 */
  align-items: center;      /* ✅ 垂直居中 */
  min-width: 200rpx;
  height: 80rpx;
  border-radius: 40rpx;
  font-size: 32rpx;
  font-weight: bold;
}
```

## 技术评估

**flex 布局优势**：
- ✅ 比 line-height 更可靠，不受字体基线影响
- ✅ 多行文本时仍能正确居中
- ✅ 不同屏幕尺寸下表现一致
- ✅ 微信小程序完全支持 flex 布局
