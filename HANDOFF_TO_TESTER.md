# 瑜伽馆课程预约系统 - 移交Tester文档

## 项目概述

瑜伽馆课程预约系统是一个完整的微信小程序解决方案，支持用户预约课程、教练管理课程、管理员管理系统三大角色功能。

## 功能模块

### 1. 用户端功能
- **首页**: 欢迎界面、推荐课程、快速入口
- **课程列表**: 课程浏览、分类筛选、搜索功能
- **课程详情**: 课程信息、时间选择、预约入口
- **预约确认**: 预约信息确认、提交预约
- **个人中心**: 用户信息、功能入口、预约概览
- **我的预约**: 预约列表、状态管理、取消预约

### 2. 教练端功能
- **我的课表**: 课程安排、时间管理、学员信息
- **学员名单**: 学员信息、预约记录、联系方式

### 3. 管理员端功能
- **课程管理**: 课程信息维护、上下架管理
- **排课管理**: 课程安排、时间调度、批量排课
- **用户管理**: 用户信息、权限管理、课时调整

## 技术架构

### 前端 (微信小程序)
- **框架**: 微信小程序原生框架
- **语言**: JavaScript, WXML, WXSS
- **组件**: 原生小程序组件

### 后端 (Node.js + Express + MySQL)
- **框架**: Express.js
- **数据库**: MySQL
- **认证**: JWT
- **安全**: CORS, Rate Limiting, Helmet

### 数据库设计 (7个表)
1. users - 用户信息
2. courses - 课程信息  
3. schedules - 排课信息
4. bookings - 预约信息
5. credit_records - 课时记录
6. coach_hours - 教练课时统计
7. 外键约束 - 数据一致性

## API接口清单

### 认证接口
- `POST /api/auth/login/wechat` - 微信登录
- `POST /api/auth/bind-phone` - 绑定手机
- `GET /api/auth/info` - 获取用户信息

### 课程接口
- `GET /api/courses` - 获取课程列表
- `GET /api/courses/:id` - 获取课程详情
- `POST/PUT/DELETE /api/courses` - 课程管理（管理员）

### 排课接口
- `GET /api/schedules` - 获取排课列表
- `GET /api/schedules/:id` - 获取排课详情
- `POST/PUT/DELETE /api/schedules` - 排课管理（管理员）

### 预约接口
- `GET /api/bookings` - 获取预约列表
- `POST /api/bookings` - 创建预约
- `PUT /api/bookings/:id/cancel` - 取消预约

## 部署信息

### 后端部署
1. 安装依赖: `npm install`
2. 配置数据库: 创建数据库并导入 schema.sql
3. 配置环境变量: 复制 .env.example 为 .env 并填入实际值
4. 启动服务: `npm start`

### 前端部署
1. 使用微信开发者工具打开 src/miniprogram 目录
2. 配置服务器域名
3. 上传代码
